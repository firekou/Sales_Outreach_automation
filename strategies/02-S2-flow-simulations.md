### 合理性閘門

```

  單則職缺，且技術棧為「ChatGPT操作」（非API整合）
  職缺描述為「協助執行」「有意願學習」= 初探者/實習性質

  公司規模22人，無法確認是否有技術決策者
  API用量治理的問題在這個規模/階段不成立
  → 不通過 ✗

Gate 3 互補性：
  他們要徵的是「學習型助理」，不是在部署LLM應用
  → 「被徵者到職就會碰到用量治理問題」不成立
  → 不通過 ✗

gate_result: HOLD → 降為 C 級觀察池，本週不打
```

```

```
動作: 加入 C 級觀察池

Dripify: 不建立 Campaign
