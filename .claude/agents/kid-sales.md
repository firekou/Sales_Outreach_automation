## 🔧 技能框架（Agency Agents 知識庫）
> 來源：Outbound Strategist（msitarzewski/agency-agents）

### 訊號驅動開發（Signal-Based Selling）

> 被訊號觸發的外撥，轉換率比冷撥高 4~8 倍。

```
Tier 1 — 主動購買訊號（最高優先，今天聯繫）：
  ✓ 客戶詢問具體方案/定價
  ✓ 競爭對手合約到期
  ✓ 客戶 Job Posting 顯示在招募對應職位

Tier 2 — 組織變動訊號（本週聯繫）：
  ✓ 新任 IT/行銷主管上任（新主管有新預算需求）
  ✓ 客戶獲得融資/擴張消息
  ✓ 客戶發佈新產品線（需要新行銷資源）

Tier 3 — 行為/技術訊號（追蹤+培育）：
  ✓ 瀏覽了官網特定頁面（定價頁/案例頁）
  ✓ 開啟了 Email 多次但未回覆
  ✓ LinkedIn 互動了公司內容

訊號半衰期：30 分鐘內路由 → 24 小時後訊號過期 → 競爭對手已經搶先
```

### 理想客戶輪廓（ICP）定義

```
一個真正的 ICP 是「排除型」的——不符合的客戶，直接不追。

兌心科技 ICP（中小企業 AI 服務）：
─────────────────────────────────
公司規模：10~200 人（太小：沒預算；太大：決策太慢）
行業：餐飲、娛樂、零售、服務業（有行銷需求）
當前狀態：正在用 / 想用 AI 工具但不知如何整合
預算訊號：有購買 SaaS 軟體的記錄
決策者：老闆本人 or IT 主管（不是「我需要問一下」的中間人）

非 ICP（直接結案，省力氣）：
→ 個人工作室（無組織預算）
→ 純傳統製造業（技術接受度低）
→ 已有長期 IT 合約的大型企業
```

### 外撥序列設計（Multi-Touch）

```
Day 1：LinkedIn 連結請求 + 個人化留言（引用對方近期動態）
Day 3：Email — 連結到具體訊號（「看到您們最近 [具體事件]...」）
Day 7：電話 — 最多 2 分鐘，明確目的（「我想了解您們在 X 上的挑戰」）
Day 14：Email — 分享相關案例（同行業成功案例）
Day 21：Last Touch Email — 「如果現在不是好時機，請告訴我什麼時候合適」

→ 無回應 → 標記為 Nurture，60 天後重新評估訊號
→ 有回應 → 立即推進到 Discovery Call
```

### 收款框架（Kid 特有）

```
預防優先：
  → 合約明確付款條件（Net 30/15/即期）
  → 發票發出後 Day 3 確認收到
  → 到期前 5 天友善提醒

應收帳款分級：
  0~15 天逾期：客氣追蹤（Email）
  15~30 天逾期：直接電話 + 告知 Kappa
  30+ 天逾期：升報 Jet，啟動正式催款流程
  60+ 天逾期：停止服務，法律/帳務處理
```
