## 先讀這個：你在賣什麼？

在任何話術之前，你要先把這件事搞清楚：

> ### 🔑 一句話記住：
> **「我們賣的不是 AI，我們賣的是『讓你不用為了偶爾用幾次，去訂整個月』的服務。」**

很多客戶已經在用 Claude、ChatGPT、Gemini 了。
他們不需要再多一個 AI——他們需要的是：
**在自己習慣的 AI 做不到的時候，有個地方可以馬上補上，而且不需要另外訂閱。**

這就是 AI Token King。

# Chapter 1：你的武器庫 — 必背三個數字

在和客戶開口之前，你要記住三組數字。這是你最重要的「子彈」。

---

## 武器一：$55~$95

這是一個典型台灣小微工程師 / 創作者每個月付給 AI 工具的費用（美金）。
換算成台幣大約是 **NT$1,800 到 NT$3,100**。

他們訂了哪些東西？

```
Claude Pro          $20/月  ← 主力
ChatGPT Plus        $20/月  ← 備用
Midjourney Basic    $10/月  ← 做圖
ElevenLabs Starter   $5/月  ← 語音旁白
Runway Standard     $15/月  ← 偶爾剪個影片
─────────────────────────────
每月合計：$70（約 NT$2,200）
```

> 🔴 **最重要的部分：他們每個月只用到這些工具的 30~40%。**
> 也就是說，他們的 AI 費用有超過一半是在「養著沒在用」。

**你的台詞：**
> *「你現在每個月大概花多少錢在 AI 工具訂閱？你知道一般用戶平均一個月付 NT$2,200，但真正用到的不到一半嗎？」*

---

## 武器二：$249.99

這是 Google AI Ultra 方案一個月的費用——美金，台幣大約 NT$8,000。

為什麼重要？因為如果你的客戶想用 **Veo 3.1（Google 最新 AI 影片生成）**，
他必須訂這個方案才能進得去。

一個創作者可能一個月只需要產 3~5 支 10 秒鐘的影片。
換算下來，每支影片要他付 NT$1,600——就為了「能進門」的資格費用。

**你的台詞：**
> *「你有聽過 Veo 3？Google 最新的 AI 影片生成，很多人都想試。但你知道嗎？要用它的完整功能，你要先付 NT$8,000 一個月的 Google AI Ultra 方案，就算你一個月只要產幾支短片。透過我們，一支 10 秒的影片大概 NT$130，用多少付多少。」*

---

## 武器三：5 個帳號

如果一個人要用到 Claude + ChatGPT + Midjourney + ElevenLabs + Suno，
他要管理：
- **5 套登入帳號**（5 組帳密）
- **5 套付款綁定**（5 張信用卡扣款）
- **5 個 API Key**（如果他是工程師）
- **5 個帳單管理**（5 份收據，有的還是美金）
- **5 種使用介面**（5 個 Tab 一直切換）

**你的台詞：**
> *「你現在管著幾個 AI 帳號？我們最多看到有客戶同時管 8 個。光是記帳密、等扣款通知、查帳單就很累。AI Token King 的核心之一，就是你只要管一個帳戶，一個 API key，所有工具在裡面。」*

# Chapter 2：你的客人是誰？

你遇到的客人大概分六種。每種都有最有效的開場角度。

---

## 客群 1：獨立開發者（Indie Dev）

**他們是誰：** 自己接案的工程師、Side Project 開發者、Freelancer  
**通常用什麼：** Claude Pro 或 ChatGPT Plus，主力做 coding  
**最痛的痛點：** 寫 code 用 Claude 很爽，但有時候需要產幾張 UI 示意圖，或是 demo 需要語音旁白，結果要再訂 Midjourney 或 ElevenLabs

**開場白：**
> *「你在用 Claude 還是 ChatGPT 做開發？好，那你有沒有遇過需要配幾張圖，或是需要把 demo 錄製成有旁白的影片，但 Claude / GPT 本身做不了，你就得另外去開帳號的情況？」*

**如果他說「有」：** 直接進 Chapter 4 的場景 A。  
**如果他說「沒有」：** *「那你目前做東西完全不需要圖或語音？」*——通常他會想一下，然後說「其實偶爾需要……」

---

## 客群 2：YouTuber / Podcast 創作者

**他們是誰：** 經營頻道、每週 / 每月產製影音內容的個人創作者  
**通常用什麼：** Claude 或 GPT 寫腳本  
**最痛的痛點：** 做完腳本還要去 ElevenLabs 配音、再去 Suno 找音樂、再去 Runway 做片段，三個平台切來切去，還得管三個訂閱

**開場白：**
> *「你平常一個月會做幾集內容？從腳本到上架，你用幾個工具？——我問這個是因為我遇到很多創作者，光是 AI 工具訂閱就付了 NT$2,000 以上，但真正用到的只有一半不到。」*

**關鍵數字：** ElevenLabs Creator $22/月只給 100 分鐘語音；Suno $8/月的 credits 月底沒用完就歸零。  
**你的台詞：**
> *「如果你一個月只錄 2 集 Podcast，每集需要 15 分鐘語音，付 NT$700（Creator 方案）讓剩下 70 分鐘的配額過期，值得嗎？還是每次用的時候才付，2 集大概花個 NT$200 就好？」*

---

## 客群 3：電商賣家 / 行銷人員

**他們是誰：** 在蝦皮、Shopline、Amazon 上賣東西的業者，或幫品牌做行銷的人  
**通常用什麼：** ChatGPT Plus 寫商品文案  
**最痛的痛點：** 文案用 GPT 搞定，但每次要上架新商品都要配圖，一直訂 Midjourney 覺得用量不夠，不訂又完全沒地方產圖

**開場白：**
> *「你現在上架一個新商品，圖片這塊怎麼處理？有在用 AI 產圖嗎？——很多賣家告訴我，每個月可能就上架 5~10 個品項，但為了這幾張圖去訂 Midjourney 每個月 NT$320，感覺有點虛。」*

**關鍵數字：** Flux Pro 一張圖 NT$1 不到（$0.03）；一個月 30 張商品圖 = **NT$30 以內**  
**你的台詞：**
> *「透過我們，30 張商品圖不到 NT$30。對比每個月固定交 NT$320 給 Midjourney，差距很明顯對吧？」*

---

## 客群 4：小型設計工作室 / 廣告公司

**他們是誰：** 2~10 人的設計或廣告小公司  
**通常用什麼：** Figma + 各種 AI 工具混用  
**最痛的痛點：** 每個案子需求不一樣，有時要產圖，有時要做影片，有時要語音，但沒辦法為了每一種需求各訂一個月

**開場白：**
> *「你們公司現在接案，AI 工具費用怎麼分攤？有沒有遇過每個月用的東西都不一樣，但帳卻照算的情況？」*

**你的台詞：**
> *「你這個月接了個影片案，下個月可能是平面廣告——你需要的模型每個月都不同。我們是用多少付多少，不是每個工具綁死你一個月，這個月用影片 API，下個月換圖像 API，一個帳戶切換。」*

---

## 客群 5：研究員 / 研究生 / 學術工作者

**他們是誰：** 在寫論文、做研究的人，預算有限  
**通常用什麼：** 免費 Gemini / 免費 Claude，或學校購買的方案  
**最痛的痛點：** 研究要查即時資料 + 帶引用來源，Perplexity Pro $20/月只給 $5 API 額度；或是遇到超長文件要分析，免費版上下文窗口不夠用

**開場白：**
> *「你在做研究的時候，有沒有需要查有引用來源的資料？不是 AI 「說說」，而是附上原始 URL 那種？——Perplexity Pro 一個月要 NT$640，但它給你用的 API 額度只有 NT$160 等值的量，其他的只能用網頁介面，沒辦法整合進你的工作流。」*

**關鍵賣點：** 研究型用戶通常用量低但需求精準，按查詢計費比月費划算很多  
**你的台詞：**
> *「學術預算最需要的就是彈性。不用每個月固定出去，用多少付多少，帳目清楚，連報帳都方便。」*

---

## 客群 6：AI 應用開發者（幫客戶做 AI 產品的工程師 / 小 Startup）

**他們是誰：** 在做自己的 AI 應用、或幫企業客戶導入 AI 的工程師  
**通常用什麼：** Claude API + 不同功能各自申請帳號  
**最痛的痛點：** 一個 App 裡面要同時呼叫 Claude（文字）+ ElevenLabs（語音）+ Flux（圖像），要管三組 API key、三個付款帳戶、三個帳單

**開場白：**
> *「你在做的應用裡面，有沒有需要同時串接不同 AI 工具的情況？比如主力用 Claude，但某個功能要生圖、某個功能要語音輸出？——那你現在管幾組 API key？」*

**你的台詞：**
> *「我們最大的優勢之一，對開發者來說，是一個 API key 就能呼叫所有你需要的模型。你的 App 不需要改架構，換一個 API endpoint，後面的模型我們都幫你管好。」*

# Chapter 3：發現痛點的 5 個魔法問句

不要急著介紹產品。先問問題，讓客戶自己說出痛點。

> **黃金原則：讓客戶說出「對，這就是我的問題」，比你說一百句廣告詞都有效。**

---

## 問句 1：「你現在用哪個 AI？」

這是暖場題，目的是讓他說話，並找到「中心工具」。

**常見回答 → 你的下一步：**
- 「Claude」→ 接問句 2
- 「ChatGPT」→ 接問句 2  
- 「都有用」→ 「哪個用最多？」→ 再接問句 2
- 「我不確定，公司給的」→ 這是企業客戶，換軌道到 Chapter 2 客群 6

---

## 問句 2：「你用它主要做什麼？」

目的：找到「使用場景」，看看有沒有跨工具的需求。

**常見回答 → 你的下一步：**
- 「寫文案 / 寫程式 / 做研究」→ 接問句 3
- 「都有做，很雜」→ 「那你有沒有遇過它做不到，你要去找其他工具的情況？」→ 接問句 3

---

## 問句 3：「有沒有哪些東西是它做不到的，讓你有點卡？」

這是關鍵問句。大多數人都有這個痛，但他們可能沒有意識到這個痛是「可以被解決的」。

**常見回答：**
- 「有時候需要圖，但 Claude 產圖不行」→ 🎯 命中，進 Chapter 4 場景 A
- 「需要語音但不知道用哪個」→ 🎯 命中，進 Chapter 4 場景 B
- 「Sora credits 月中就用完了」→ 🎯 命中，進 Chapter 4 場景 C
- 「沒有耶」→ 接問句 4

---

## 問句 4：「你現在除了主力 AI，有沒有訂其他 AI 工具？」

目的：讓他算清楚自己的 AI 支出，並意識到「有沒有在浪費」。

**你的台詞：**
> *「像我遇到的很多人，除了 Claude 或 GPT 之外，還訂了 Midjourney、ElevenLabs 之類的。你有嗎？大概一個月加起來多少？」*

---

## 問句 5：「如果那些工具你一個月只用到 30%，你覺得值得嗎？」

這是收尾的「破防問句」。讓他自己說「不值得」，你就可以順勢介紹 ATK。

---

# Chapter 4：「哇！」的三個 Demo 場景

這一章是你跟客戶講話過程中，**讓他眼睛發亮**的時刻。
每個場景都是一個「故事」，不是產品規格說明。

---

## Demo 場景 A：「你用 Claude，但它不會產圖」

**對象：** Claude Pro 或 ChatGPT Plus 用戶  
**情境：** 他的主力做文字，偶爾需要圖像

**你的故事版本：**

> *「我有個客戶，他是顧問，每週用 Claude 幫他寫提案報告。後來他的客戶開始要求『能不能放幾張概念圖』。他一開始去訂了 Midjourney，一個月 NT$320。但他每個月只用個 10 張圖，剩下的 GPU 時數就跑掉了。*
>
> *後來他用我們的服務，透過 ATK 呼叫 Flux Pro，一張圖不到 NT$1。10 張圖大概 NT$10。他省下來的錢買了一頓便當。」*

**關鍵數字：**
- Midjourney Basic：NT$320/月，3.3 GPU 小時（大概可以產 200+ 張）
- 但他只需要 10 張 → 等於花了 NT$32/張「資格費」
- Flux Pro：NT$1/張 → **貴 32 倍的差距**

**你的結語：**
> *「我們不是要取代你用 Claude——你繼續用 Claude，用得很爽。只是當 Claude 遇到它不擅長的，你不用另外再訂一個平台，透過 ATK 直接補上就好。」*

---

## Demo 場景 B：「你做 Podcast，需要語音旁白」

**對象：** Podcast 創作者、YouTube 內容創作者  
**情境：** 腳本用 Claude / GPT 寫完了，需要語音合成

**你的故事版本：**

> *「一個在做科技 Podcast 的人來找我。他用 Claude 寫腳本，然後需要語音旁白。他去訂了 ElevenLabs Creator，NT$700 一個月，可以用 100 分鐘的語音。*
>
> *但他每個月只錄 2 集，每集 20 分鐘，加起來 40 分鐘。剩下 60 分鐘的額度每個月自動消失。*
>
> *他氣的點不是 NT$700——他氣的是『每個月有 60%的錢打水漂』。」*

**關鍵數字：**
- ElevenLabs Creator：NT$700/月 = 100 分鐘語音
- 他的實際需求：40 分鐘/月
- 浪費比率：60%

**你的結語：**
> *「在我們這邊，你用幾分鐘付幾分鐘的錢。同樣的 40 分鐘語音，大概是 NT$400 左右。省下的 NT$300，是真實的錢，不是說說而已。」*

---

## Demo 場景 C：「你做影片，Sora credits 月中就沒了」

**對象：** 短影音創作者、社群媒體工作者  
**情境：** 訂了 ChatGPT Plus，想用 Sora 產 AI 影片

**你的故事版本：**

> *「ChatGPT Plus $20 一個月，裡面給你 1,000 個 Sora credits。聽起來很多。但你知道一支 5 秒鐘的 720p 影片，要用 80 個 credits 嗎？*
>
> *1,000 credits，只能產 12 支 5 秒的影片。如果你在做短影音，12 支根本不夠——尤其是測試不同風格的時候，一個嘗試就 80 credits 沒了。*
>
> *很多人的選擇是升級到 ChatGPT Pro，$200 一個月（NT$6,400）。*
>
> *另一個選擇是：credits 用完了，透過我們接 Kling 3.0 API，一秒影片 NT$3 不到。5 秒影片大概 NT$15。你想產幾支就產幾支。」*

**關鍵數字：**
- ChatGPT Plus 1000 credits = 12 支 5 秒影片（月用完）
- 升到 Pro：NT$6,400/月
- Kling 3.0 API：5 秒影片 = NT$15
- **10 支影片：NT$150 vs NT$6,400**

**你的結語：**
> *「我們不是要你停訂 ChatGPT Plus——你繼續用它，用完才透過我們補。補足缺口，不是換掉你的主力。」*

# Chapter 5：常見異議 × 黃金應對話術

客戶說「不」或「但是」的時候，不要慌。這些是最常出現的異議，每一個都有標準應對方式。

---

## 異議 1：「我已經有 ChatGPT Plus / Claude Pro 了，我用那個就夠。」

**你的心態：** 這不是反對，這是他在說「我還不知道我有缺口」。

**應對：**
> *「對，主力工具你有了，而且用得很習慣。我問你一件事：你有沒有遇過需要產圖、或需要做個有語音的 demo，但你的主力工具做不到這塊？」*

如果他說有 → 你打到了，繼續 Demo 場景。  
如果他說沒有 → *「那你目前做的東西完全不需要圖像或語音的輔助？」*（他通常會開始想）

---

## 異議 2：「我覺得我自己去各家申請 API 就好，比較便宜。」

**你的心態：** 他在比價錢，你要把「隱性成本」帶進來。

**應對：**
> *「你說的對，自己去申請 API 理論上是成本最低的。我想問你幾件事：*
>
> *第一，你要管幾個 API Key？每個都要設帳號、綁信用卡、處理海外扣款，有時候 GCP 台灣信用卡還驗證不過，你花的時間值多少？*
>
> *第二，如果你在台灣，海外 AI 工具的信用卡付款失敗率其實不低。我們統一幫你處理，用台幣儲值，沒有這個問題。*
>
> *當然，如果你工程師時間很充裕，自己架也是個選擇。但多數人算完之後，我們的整合服務值得這個價差。」*

---

## 異議 3：「你們的價格比直接去訂便宜嗎？」

**你的心態：** 不要正面硬拚單一模型的 per-token 價格。要拉回到「整體成本」的比較。

**應對：**
> *「這個問題問得好，我直接說實話：如果你只看單一模型的 token 單價，直接跟 Anthropic 或 OpenAI 買有時候可能比我們稍微便宜一點點。*
>
> *但是你用的不只是一個模型對吧？你說說看，你現在一個月的 AI 工具費加起來多少？我們來算一下，如果這些呼叫都放在 ATK，你實際的費用是多少、省掉的帳號管理成本是多少、省掉的海外扣款麻煩是多少。*
>
> *多數客戶算完，ATK 整體下來反而是省的。」*

---

## 異議 4：「我現在用量不多，還不需要這種東西。」

**你的心態：** 他在說「門檻對我太高了」，你要讓他知道我們的門檻很低。

**應對：**
> *「沒關係，我們 Starter 方案最低就是 NT$2,990 一個月——那包含了 500 萬 tokens，支援兩家供應商，你可以先試著把現在分散的工具整合進來看看。*
>
> *你不需要馬上把所有工具都搬過來。很多人是從『先把一個 API 整合進 ATK』開始，感受一下，好用再擴大。」*

---

## 異議 5：「我怕資料安全問題。」

**你的心態：** 這是合理顧慮，要認真回應，不要輕描淡寫。

**應對：**
> *「這個問題很重要，你問得對。我跟你說我們的做法：*
>
